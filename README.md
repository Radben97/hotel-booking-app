# Holidays 🏨

A complete hotel booking application built using TypeScript, React, Node.js (Express), MongoDB, and Tailwind CSS.

The application is fully developed and running. Users can search for hotels, filter results, view hotel details, make bookings, and complete payments through the integrated payment system.

## 🌐 Live Website

The application is deployed and available here:

👉 **[Click here to open Holidays](https://hotel-booking-app-267e.onrender.com)**

*Note: As this is hosted on Render's free tier, the website may take 30–60 seconds to load initially if the server has been idle.*

---

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Image Storage:** Cloudinary
* **Payments:** Razorpay (Test Payment Integration)
